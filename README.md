# dijital-sinif
Bu proje, okulda ölçme-değerlendirme süreçlerini daha adil, güvenli ve erişilebilir hale getirmek amacıyla geliştirilen Dijital Sınav Sınıfı tasarımını içermektedir. Jones (2005) Teknoloji Planlama Modeli doğrultusunda planlanan çalışma; ihtiyaç analizi, uygulama ve değerlendirme aşamalarını bütüncül bir yaklaşımla ele almaktadır.
